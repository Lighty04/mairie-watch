# Task 2.1: Advanced Alert Rules UI

## Context
The current alert rules form is basic. Need advanced filters UI in `app/templates/alerts.html`.

## Requirements
Update `app/templates/alerts.html` to include:
1. **Amount threshold filter**: input field + dropdown (>, <, >=, <=, =)
2. **Arrondissement multi-select**: Checkboxes for 1-20 (grouped in rows of 5)
3. **Boolean query builder**: Textarea with helper text showing syntax:
   ```
   Exemples:
   subvention AND (sport OR culture)
   "marché public" NOT "marché forain"
   ```
4. **Time window**: Dropdown (last 7 days, last 30 days, all time)
5. **Exclusion keywords**: Textarea for keywords to exclude

The form should show/hide advanced fields with a toggle "Mode avancé".

## Acceptance
- HTML form renders correctly
- Toggle shows/hides advanced fields
- Multi-select arrondissements works
- Form submits to existing `/api/alert-rules/advanced` endpoint
- Mobile responsive
