# Task 1.1: Keyword Context Extraction for Alerts

## Context
The `generate_alert_email()` in `app/advanced_alerts.py` finds matching keywords but the context extraction is basic. Need to:
1. Extract 2 sentences before and after each keyword match
2. Highlight the keyword in the context
3. Return a list of `{keyword, context_before, match, context_after}`

## Current Code (in app/advanced_alerts.py)
```python
# Find matching keywords with context
highlights = []
if rule.keywords:
    text = decision.raw_text or ""
    for kw in rule.keywords:
        # Find keyword in text
        kw_lower = kw.lower()
        idx = text.lower().find(kw_lower)
        if idx >= 0:
            start = max(0, idx - 100)
            end = min(len(text), idx + len(kw) + 100)
            context = text[start:end]
            # Highlight keyword
            highlighted = context.replace(
                text[idx:idx+len(kw)],
                f"**{text[idx:idx+len(kw)]}**"
            )
            highlights.append(highlighted)
```

## Requirements
- Extract exactly 2 sentences before and 2 sentences after the match
- A sentence ends with `.`, `!`, or `?`
- If fewer than 2 sentences exist before/after, return what's available
- Highlight the matched keyword with `<mark>` HTML tags
- Return structured data: list of dicts with `keyword`, `before`, `match`, `after`
- Handle multiple occurrences of the same keyword (return first 3)

## Acceptance
- Works with the existing `generate_alert_email()` function
- Sentence splitting handles French text correctly (espaces insécables, etc.)
- Returns max 3 highlights per keyword
